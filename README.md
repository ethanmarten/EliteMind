# 🏛️ EliteMind - Elite Knowledge Engine

EliteMind هو محرك معرفة نخبوي يعتمد على الذكاء الاصطناعي لتحليل المستندات (PDF، DOCX، TXT)، روابط الويب، وفيديوهات يوتيوب، والإجابة على الأسئلة بدقة وسرعة فائقة باستخدام **Groq** ونموذج **Llama-3-70B**.

## ✨ المميزات
- واجهة Black & Gold فاخرة مع تأثير زجاجي.
- سرعة خارقة بفضل Groq.
- دعم متعدد المصادر (ملفات، روابط، يوتيوب).
- Embeddings مجانية محلية بدون تكاليف.
- إمكانية تصدير الإجابة كـ PDF بنقرة زر (عبر الطباعة).
- جاهز للنشر على Streamlit Cloud.

## 🚀 التشغيل المحلي
```bash
pip install -r requirements.txt
streamlit run app.py
